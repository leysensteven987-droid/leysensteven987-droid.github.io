# leysensteven987-droid.github.io

User-site landing page for **kage-gumi.com** (影組 — Shadow Crew Protocol).

This repo is the GitHub Pages *user site*. Because the custom domain `kage-gumi.com`
is attached here, every project Pages site on this account is automatically served
underneath it at its repo path:

| URL | Source repo |
|---|---|
| `kage-gumi.com` | this repo (`index.html`) |
| `kage-gumi.com/kage-gumi-presentation/` | `kage-gumi-presentation` |
| `kage-gumi.com/kage-gumi-keynote/` | `kage-gumi-keynote` |

The decks need no per-repo custom domain — they inherit this one (their
`username.github.io/...` URLs redirect here). Do **not** add a `CNAME` file to the
deck repos or the inheritance breaks.

Brand: dark `#0e1a18`, teal `#00ffe0`, crimson `#e8365d`, DM Sans + Noto Serif JP,
closing punchline "The crew presents itself."
