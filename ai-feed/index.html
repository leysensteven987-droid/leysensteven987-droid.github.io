JadePuffer: The First Documented Fully Autonomous AI Ransomware - No Human Operator Needed

Sysdig's threat research team published a full analysis in early July 2026 of JADEPUFFER, the first documented end-to-end autonomous ransomware attack run by an LLM agent. The agent handled every stage without human guidance - reconnaissance, credential theft, lateral movement, privilege escalation, persistence, and encryption. Its code was saturated with natural-language annotations explaining why each step was taken, a pattern unique to LLM-generated code that no human operator would add to a disposable exploit payload.

What's moving
  • JadePuffer gained initial access by exploiting CVE-2025-3248, an unauthenticated remote code execution flaw in Langflow, a widely used open-source framework for building LLM applications - raising the attack surface risk of the entire production LLM tooling ecosystem and every team running Langflow in production
  • The LLM agent recovered from a failed login in 31 seconds by self-diagnosing and retrying, executing the full lateral movement phase in minutes rather than the hours typical of human operators - demonstrating real-time adaptive problem-solving that previous script-based ransomware could not achieve
  • The attack establishes that ransomware operators no longer need deep expertise: the LLM chained reconnaissance, credential theft, lateral movement, persistence, and destruction as a coordinated sequence - lowering the skill floor for sophisticated attacks to near zero and requiring enterprises to update their threat models immediately
